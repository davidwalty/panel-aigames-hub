# Panel AIGames Hub

## 项目简介

Panel AIGames Hub 是一个用于归档和发布多个独立 HTML 页面的仓库。这些页面通常围绕 AI 游戏、交互式小工具或前端演示内容，不针对任何特定网站或域名，旨在提供一份干净、可直接运行的页面集合。

## 目录说明

- `pages/` — 存放所有独立 HTML 页面，每个页面为一个文件或一个子目录。
- `assets/` — 页面共用的资源文件，如图片、样式、脚本等。
- `docs/` — 项目相关文档或说明（可选）。
- `README.md` — 本文件。

## 页面归档说明

每个页面均为自包含的 HTML 文件，尽量不依赖外部资源（或已内联）。页面之间相互独立，可直接在浏览器中打开运行。

如需新增页面，请遵循以下约定：

1. 页面文件统一放入 `pages/` 目录。
2. 文件名使用小写英文，以连字符分隔（如 `tictactoe.html`）。
3. 如需引用共用资源，请使用相对路径指向 `assets/` 目录。

## 维护说明

- 本仓库仅用于归档与展示，不提供在线服务或动态部署。
- 页面内容可能不定期更新或清理，不保证向后兼容。
- 欢迎提交 Issue 或 Pull Request，但请确保内容与仓库定位一致。

## 许可

除非另有声明，本仓库内容仅用于学习与参考，请勿用于商业用途。
